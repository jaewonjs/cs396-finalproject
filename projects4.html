Trip Planner

Spring 2020

Problem Description: Have you ever been to a new city without knowledge where points of interest are? This seems to be a common problem for tourists even today. To combat this, I created a system to find the closest points of interest by category or the shortest path to a destination to make sure you don't have to drive around aimlessly for hours!

Tech Stack: This project made use of several fundamental data structures such as minheaps, graphs, and dictionaries to organize a city's road networks and landmarks. Then by using several minimization algorithms I was able to achieve the objectives and return them in an orderly fashion to the user.
